Datos del cliente

Esta ventana de la aplicación tiene como función recoger datos del usuario para poder formular la reserva.

Está divido en los siguientes apartados:

  • Nombre :Nombre del cliente que está efectuando la reserva.
  • Apellidos : Apellidos del cliente que está reservando.
  • DNI o pasaporte : DNI para clientes españoles y DNI para clientes extranjeros.
  • Telefono : Teléfono a través del cual contactar al cliente en posible caso de incidencia.
  • Email : Sirve para contactar al cliente y también para planes de fidelización y campañas de mailing.
  • Observaciones: Posibles indicaciones especiales sobre la reserva.

También se incluyen los botones de Atrás para volver a Vista de reservas y Siguiente para proseguir en el proceso de reserva.
